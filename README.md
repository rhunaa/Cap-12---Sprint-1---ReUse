# ReUse

## Sobre o projeto

O ReUse é um aplicativo mobile voltado ao consumo consciente e à reutilização de produtos.

O objetivo da plataforma é conectar pessoas interessadas em doar, trocar ou reutilizar itens, ajudando a reduzir desperdícios.

---

## Contexto da Sprint

Projeto desenvolvido para a Sprint 1 da FIAP utilizando React Native.

Nesta etapa, o foco foi criar a tela inicial do aplicativo e apresentar a proposta do ReUse.

---

## Problema abordado

Muitos produtos ainda úteis são descartados diariamente, aumentando o desperdício e a produção de lixo.

O ReUse busca incentivar a reutilização de produtos de forma simples e acessível.
